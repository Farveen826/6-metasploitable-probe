# Metasploitable Vulnerability Probe

## Objective
To perform vulnerability scanning and enumeration on Metasploitable VM.

## Tools Used
- Nessus Essentials
- Nikto
- Netcat

## Artifacts
- Nessus vulnerability scan report (PDF)
- Nikto web vulnerability report (HTML)
- Netcat banner grabbing screenshot

## Conclusion
The Metasploitable VM contains multiple critical and high vulnerabilities due to outdated and misconfigured services.
